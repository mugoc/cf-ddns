# cf-ddns-1
**使用方法：**
```bash
wget https://github.com/mugoc/cf-ddns/raw/main/install.sh && chmod +x install.sh && bash install.sh
```
修改/etc/cf-ddns-1-config.json
```bash
{
  "Zone_ID": "",		    # 主域名ID
  "Email": "",		      # CF邮箱账户
  "Key": "",			      # Global Key
  "Records_NAME": ""		# 需要更新IP域名
}
```
